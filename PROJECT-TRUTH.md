# BeautyOS AI v2: PROJECT TRUTH (v1.0)

Это — единственный и окончательный источник правды для всех агентов и разработчиков проекта. Все остальные файлы в корне (AGENTS.md, ANTIGRAVITY.MD и др.) объявляются недействительными и подлежат удалению.

## 1. Ядро Системы
- **Проект**: BeautyOS AI v2 — студия премиального AI-маркетинга.
- **Эстетика**: Luxury Minimal (чистые линии, профессиональная ретушь, отсутствие визуального мусора).
- **Стек**: React 19 + Vite, Tailwind 4, Supabase (бессерверные сессии), Vercel.

## 2. AI Конвейер (Целевая архитектура)
- **Модели**: Gemini 1.5 Pro (Анализ), Imagen 4 (Генерация/Ретушь).
- **Процессор**: Unified Processor (единая точка входа для Бота и Дашборда).
- **Алгоритм**: Stacking v62 (Анализ -> AI_SEED -> Ретушь -> Рендер).
- **Outpainting**: Использование цветовой заливки #1A1A1A для исключения артефактов Imagen.

## 3. Рабочий Режим (Дисциплина)
- **Цикл RARV**: Reason (Рассуждение) -> Act (Действие) -> Reflect (Анализ) -> Verify (Верификация).
- **Принципы**: "One feature at a time", "Verify before Done", "Think before Code".
- **Язык**: Русский (по умолчанию для всех отчетов и артефактов).

## 4. Инструментарий (MCP Статус)

- **notebooklm**: Подключен (Архитектура и База знаний).
- **chrome-devtools**: Подключен.
