# Системный Архитектор (Senior Architect)

Полный набор инструментов системного архитектора с современными практиками проектирования.

## Основные возможности

Этот навык предоставляет три ключевые функции через автоматизированные скрипты (для локального использования агентом):

```bash
# Скрипт 1: Генератор архитектурных диаграмм
python scripts/architecture_diagram_generator.py [options]

# Скрипт 2: Архитектор проекта (анализ структуры)
python scripts/project_architect.py [options]

# Скрипт 3: Анализатор зависимостей
python scripts/dependency_analyzer.py [options]
```

## Ключевые области

### 1. Проектирование систем
- Определение архитектурных паттернов (Microservices, Layered, Event-Driven).
- Выбор технологического стека.
- Проектирование схем баз данных (PostgreSQL, Supabase).

### 2. Анализ и оптимизация
- Глубокий аудит структуры проекта.
- Поиск узких мест в производительности.
- Рекомендации по масштабируемости.

### 3. Документация (RFC/ADR)
- Написание документов о проектных решениях (Architectural Decision Records).
- Генерация технических спецификаций.

## Технологический стек
- **Frontend:** React, Next.js, Telegram Mini App API.
- **Backend:** Node.js, TypeScript, Express, GraphQL.
- **Инфраструктура:** Vercel, Supabase, GitHub Actions.

## Лучшие практики
- **Качество кода:** Следование SOLID, минимизация технического долга.
- **Безопасность:** Валидация данных, безопасная аутентификация.
- **Масштабируемость:** Stateless-архитектура, кэширование, эффективные запросы.

---
*Примечание: Python-скрипты в директории `scripts/` являются вспомогательными инструментами агента для анализа локального кода.*
