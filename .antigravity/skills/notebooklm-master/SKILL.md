# NotebookLM Master Skill: The Bi-Cameral Mind (Мозг и Руки)

## Goal (Цель)
Использовать NotebookLM как «Второй мозг» (хранилище знаний, архитектуры и долгосрочной памяти) для разгрузки оперативной памяти Antigravity, а также как мощный исследовательский инструмент для анализа кодовой базы и генерации контента.

## Instructions (Инструкции по использованию)

### 1. Анализ и Создание Базы Знаний (Project Snapshot)
Если пользователь просит оптимизировать проект, сохранить прогресс или создать документацию:
1. **Анализ:** Проведи полный аудит текущей кодовой базы.
2. **Генерация локальных файлов:** Создай в корне проекта папку `docs/` и сгенерируй:
   - `context.md`: Описание бизнес-логики и текущего состояния.
   - `architecture.md`: Технический стек и связи между компонентами.
   - `lessons_learned.md`: Список решенных проблем и предотвращение старых ошибок.
3. **Ingestion:** Создай новый блокнот в NotebookLM "[Имя проекта] — Knowledge Base" через MCP и загрузи туда эти файлы.

### 2. Умная петля улучшений (The Smart Loop)
При запросе на улучшение или рефакторинг:
- Выгрузи целевой код в NotebookLM.
- Сделай запрос: "Проанализируй код на основе сохраненной архитектуры и найди технический долг или уязвимости".
- Получи отчет и примени исправления в коде Antigravity.

### 3. Глубокое исследование (Deep Research)
Для изучения новых технологий или конкурентов:
- Используй NotebookLM для анализа внешних источников (URLs, PDF).
- Создай тематический блокнот и выдай пользователю детальный план внедрения (TL;DR).

### 4. Генерация артефактов (Artifact Generation)
По запросу создавай через NotebookLM:
- **Audio Overviews:** Аудио-подкасты с обсуждением архитектуры.
- **Study Guides / Briefings:** Справочники по проекту.
- Сохраняй результаты в локальную папку `outputs/`.

### 5. Очистка контекста и Перенос (Context Handover)
Перед завершением длинной сессии:
- Создай максимально детальное **Summary** всей работы.
- Подготовь Git Commit с описанием изменений.
- Инструктируй пользователя использовать это Summary для открытия нового чистого чата.

## Constraints (Ограничения)
1. **Zero Hallucinations:** Опирайся СТРОГО на данные из NotebookLM.
2. **Context Economy:** Извлекай только суть, не перегружая чат Antigravity лишним текстом.
3. **Security:** Всегда проверяй статус подключения MCP-сервера перед началом работы.
