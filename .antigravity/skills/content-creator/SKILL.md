# Создатель контента (Content Creator)

Профессиональный анализ голоса бренда, SEO-оптимизация и платформенные фреймворки контента.

## Когда использовать

Используйте этот навык при написании постов в блог, создании контента для соцсетей, разработке голоса бренда, SEO-оптимизации или планировании контент-календаря.

## Ключевые слова
создание контента, посты в блог, SEO, голос бренда, соцсети, контент-календарь, маркетинговая стратегия, контент-маркетинг, консистентность бренда.

## Быстрый старт

### Для разработки голоса бренда
1. Запустите `scripts/brand_voice_analyzer.py` на существующем контенте для определения базы.
2. Изучите `references/brand_guidelines.md` для выбора атрибутов голоса.
3. Применяйте выбранный голос ко всему новому контенту.

### Для создания контента в блог
1. Выберите шаблон из `references/content_frameworks.md`.
2. Исследуйте ключевые слова для темы.
3. Напишите текст, следуя структуре шаблона.
4. Запустите `python scripts/seo_optimizer.py [файл] [ключевое-слово]` для оптимизации.

### Для контента в соцсети
1. Проверьте лучшие практики платформ в `references/social_media_optimization.md`.
2. Составляйте график публикаций, используя `assets/content_calendar_template.md`.

## Ключевые скрипты

### brand_voice_analyzer.py
Анализирует текст на характеристики голоса, читаемость и последовательность.
**Использование**: `python scripts/brand_voice_analyzer.py <файл>`

### seo_optimizer.py
Анализирует контент на предмет SEO-оптимизации и дает рекомендации.
**Использование**: `python scripts/seo_optimizer.py <файл> [ключевое_слово]`

## Рекомендации по качеству
- SEO-показатель выше 75/100.
- Читаемость соответствует целевой аудитории.
- Единый голос бренда во всем тексте.
- Наличие четкого призыва к действию (CTA).
